Projeto Formulário de Cadastro

Este projeto foi desenvolvido para testar validações em JavaScript, garantindo a integridade e segurança dos dados inseridos.

Regras para preenchimento:

  • Todos os campos são obrigatórios.
  • O nome de usuário deve conter apenas letras e/ou números.
  • O nome de usuário deve ter entre 3 e 12 caracteres.
  • A senha deve ter entre 6 e 12 caracteres.
